### Описание проекта  
Имеется файл формата CSV, каждая строка в этом наборе данных представляет собой молекулу.  Первый столбец содержит экспериментальные данные, описывающие реальную биологическую реакцию: молекула вызвала эту реакцию (1) или не вызвала (0). Остальные столбцы представляют собой молекулярные дескрипторы (от d1 до d1776) — рассчитанные свойства, которые могут отражать некоторые характеристики молекулы, например размер, форму или элементный состав. Матрица дескрипторов была нормализована.
